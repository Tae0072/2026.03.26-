# DOM 요소 동적 추가/삭제

바닐라 JavaScript로 구현한 DOM 요소 동적 추가/삭제 실습 프로젝트입니다.

## 프로젝트 구조

```
├── index.html   # 마크업 구조
├── style.css    # 스타일 정의
└── script.js    # 동작 로직
```

## 주요 기능

- **항목 추가**: 텍스트 입력 후 `추가` 버튼 클릭 또는 `Enter` 키 입력으로 항목 생성
- **항목 삭제**: 각 항목 우측 `✕` 버튼 클릭으로 해당 항목 제거
- **빈 값 방지**: 빈 입력 시 경고 알림 표시
- **빈 컨테이너 안내**: 항목이 없을 때 안내 메시지 표시 (CSS `::before` 활용)
- **페이드인 애니메이션**: 항목 추가 시 부드러운 등장 효과

## 실행 방법

별도 빌드 과정 없이 `index.html`을 브라우저에서 바로 열어 사용합니다.

## 기술 스택

- HTML5
- CSS3 (Flexbox, Keyframe Animation)
- Vanilla JavaScript (DOM API)
